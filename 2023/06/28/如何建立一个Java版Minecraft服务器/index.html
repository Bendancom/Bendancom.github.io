开启防火墙

查看Minecraft服务器端口，默认为25565

开放对应端口的IPv4与IPv6访问
目前内网穿透有两种方式

  • Frp
  • P2P

Frp可能需要付费(有些有限流限额的免费)，但仅需服务端进行配置，客户端无需繁琐操作
P2P完全免费，但服务端与客户端都要进行专门的配置

注：开放IPv6可以免去内网穿透，但是并不是所有玩家都支持IPv6访问，可能受到路由器、光猫、运营商等的拦截
